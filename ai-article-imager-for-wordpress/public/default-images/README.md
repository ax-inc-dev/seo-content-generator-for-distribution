# デフォルト画像フォルダ

このフォルダに配置した画像が、ベース画像として自動的に使用されます。

## 画像の配置方法

1. このフォルダに画像ファイル（jpg, png, webp等）を配置してください
2. ファイル名にキーワードを含めることで、適切なH2セクションに自動マッチングされます

## ファイル名の例

### 推奨される命名例
- `メリット画像.jpg` - メリット系のH2に使用
- `デメリット・リスク.png` - デメリット系のH2に使用
- `概要説明.jpg` - 概要・基本系のH2に使用
- `比較イラスト.png` - 比較系のH2に使用
- `解決策の図.jpg` - 解決策系のH2に使用
- `FAQ用画像.png` - FAQ系のH2に使用
- `料金プラン.jpg` - 料金系のH2に使用

## 注意事項

- ファイル名は日本語でもOKです
- 複数のキーワードを含んでもOKです（例：「メリット・利点・強み.jpg」）
- 番号を付けてもOKです（例：「01_メリット.jpg」）
- ファイルサイズは適度に圧縮してください（推奨: 500KB以下）
