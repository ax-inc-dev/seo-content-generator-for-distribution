APIとは？
© AXinc

APIとは「Application Programming Interface」の略で、
あるサービスやシステムの機能を外部から呼び出す“窓口”です。
 天気予報API → 「今日の天気データ」を取得できる
 地図API → 「地図を表示する」「ルートを検索する」
アプリやサービス同士が直接データをやり取りできるようになり、
自分で一から仕組みを作らなくても、既存の機能を組み合わせて利
用できます。
「API」とは？
システムの機能を借りるための窓口

APIは課金が必要なケースもあります
© AXinc
APIとは？
例えば、ChatGPTなどのLLMをAPIで使用する際は
課金が必要です。課金が必要ないAPIもあるので、利用時に必ず調べましょう。

APIキーは漏らさないように！
© AXinc
APIとは？
APIを利用する時は、APIキーというものが発行されます。
これは外部サービスと連携する際に必要なパスワードのようなものです。
不正利用や情報流出のリスクになるので絶対に外部に漏らさないでください。
※キーを漏らさないように注意

API連携はコードエラーが起きやすいので注意
© AXinc
APIとは？

GmailのAPI呼び出しでエラー！
(NotionのAPI仕様が呼ばれていた)
何度もエラーが起きる「Gmail API リファレンス」で検索公式リファレンスをコピーして貼り付け
LLMは様々なWeb上の情報を学習しており、別のAPI情報を呼び出す可能性もあります。
ハルシネーションが原因なので、正しい情報である公式の“APIリファレンス”を貼り付けると
ある程度回避できます。
