# ⚠️ 重要：アプリが動かない時は

## 🚨 「表示されない」「動かない」場合の解決法

### すぐに実行するコマンド：
```bash
./start.sh
```

これで全部起動します！

---

## もしstart.shが動かない場合：

### 手動で起動：
```bash
# 1. 古いプロセスを終了
pkill -f vite
pkill -f scraping-server.js

# 2. サーバーを起動（新しいターミナルで）
cd server && node scraping-server.js

# 3. アプリを起動（別の新しいターミナルで）
npm run dev
```

### アクセスURL：
- アプリ: **http://localhost:5176/**
- 検索が動かない場合は、サーバーが起動してるか確認

---

## 🔍 よくある問題

### 「SearchAPIが動かない」
→ Puppeteerサーバー（ポート3001）が起動していない
→ `./start.sh` を実行

### 「ページが真っ白」
→ Viteサーバーが起動していない
→ `./start.sh` を実行

### 「記事生成に失敗」
→ 両方のサーバーが必要
→ `./start.sh` を実行

---

## 📝 覚えておくこと

**困ったら `./start.sh` を実行すれば解決！**
